# 🛍️ TechStore Chile

Proyecto frontend de e-commerce desarrollado como parte del Módulo 2.

## 🚀 Tecnologías utilizadas

- HTML5 semántico
- Bootstrap 5
- JavaScript básico
- Git y GitHub

## 📄 Funcionalidades

- Página de inicio con productos (cards)
- Página de detalle de producto
- Carrito simulado
- Contador dinámico en navbar
- Diseño responsive

## 📁 Estructura

- index.html → página principal
- producto.html → detalle de producto
- carrito.html → carrito
